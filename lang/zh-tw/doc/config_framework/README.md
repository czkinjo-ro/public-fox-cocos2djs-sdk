## Framework設定的詳細

點擊編譯對象，「Build Phases」→選擇「Link Binary With Libraries」。點擊「＋」按鈕，請選擇各Framework。

![Framework設定01](./img01.png)

請把下面的Framework追加到開發項目裡。
